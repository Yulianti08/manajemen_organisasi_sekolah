# Sistem Manajemen Anggota Organisasi Sekolah

Deskripsi:
Website untuk pengelolaan organisasi sekolah yang memungkinkan siswa melihat informasi organisasi, kegiatan, dan melakukan pendaftaran anggota secara online.

Framework:
- Node.js
- Express.js
- MongoDB
- EJS

Cara Menjalankan:
1. Install dependency
   npm install

2. Jalankan aplikasi
   node app.js

3. Buka browser
   http://localhost:3002

## Cara Menggunakan Aplikasi

### User (Siswa)
1. Buka aplikasi melalui browser.
2. Pilih menu Register untuk membuat akun.
3. Login menggunakan email dan password yang telah didaftarkan.
4. Masuk ke dashboard utama.
5. Melihat daftar organisasi yang tersedia.
6. Memilih organisasi yang diminati.
7. Melihat detail organisasi dan kegiatan yang tersedia.
8. Mengisi formulir pendaftaran organisasi.
9. Melihat status pendaftaran pada menu Pendaftaran Saya.
10. Mengelola data profil pada menu Profil.

### Admin
1. Login menggunakan akun admin.
2. Masuk ke dashboard admin.
3. Mengelola data organisasi.
4. Mengelola data kegiatan.
5. Melihat data pengguna.
6. Melihat data pendaftaran.
7. Menerima atau menolak pendaftaran anggota.

#Ketua Organisasi
1. Login menggunakan akun ketua organisasi.
2. Melihat daftar calon anggota yang mendaftar.
